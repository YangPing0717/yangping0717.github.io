出版
研发

好学教育研究院，提供专为 0-12 岁孩子设计之教材、教案。目前针对 0-2y / 2-3y / 3-6y / 6-12y 等年龄段孩子，设计符合医教保理念教材，并产出提供给教师们相应之教案，截至目前已累积了超过 400 本教材、上千份的教案。

培训

除了提供教材教案外，更提供各式形式的专家培训活动，为使医学、教育、保育概念更明确实践，培训内容之设计更是好学教育研究员的核心服务内容。

品牌规划
